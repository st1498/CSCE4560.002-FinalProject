# Cybermax E-Commerce Project

## Project Structure

```text
CSCE4560.002-FinalProject/
└─backend/
  └─migrations
  └─templates
    ├─__init__.py
    ├─admin.py
    ├─apps.py
    ├─forms.py
    ├─models.py
    ├─tests.py
    ├─urls.py
    ├─views.py
└─cybermax/
  ├─__init__.py
  ├─asgi.py
  ├─settings.py
  ├─urls.py
  ├─wsgi.py
└─.gitignore
└─README.md
```

## How to run the project

```bash
python manage.py runserver
```
